Gestione password
=================

In questa schermata è possibile, per il dipendente, modificare la propria password di accesso al sistema ePAS.

.. figure:: _static/images/cambiaPassword.png
   :scale: 40
   :align: center
   
   Schermata di cambio password
   
Come visibile dalla schermata, per il cambio password è richiesto di inserire la vecchia password, di specificarne una nuova e, infine, di riscriverla per motivi di sicurezza.
